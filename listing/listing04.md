Что выведет программа? Объяснить вывод программы.

```go
package main

func main() {
	ch := make(chan int)
	go func() {
		for i := 0; i < 10; i++ {
			ch <- i
		}
		// решение: 
		// добавить
		// close(ch)
	}()

	for n := range ch {
		println(n)
	}
}
```

Ответ:
```
Выведет от 0 до 9, после произойдет deadlock, 
т.к. range будет продолжать ждать значение, 
а не в main горутине запись закончится.

Хорошим решением будет закрыть канал после цикла for  
Для того чтобы оператор range завершил считывание корректно.
```
